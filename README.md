# Weather Diary (Дневник погоды)

Автор: Грусков Илья

Описание:
Это простое GUI-приложение для ведения дневника погоды. Позволяет добавлять, просматривать, фильтровать и сохранять записи о погоде в JSON-файл.

Как запустить:
1. Убедитесь, что установлены Python и библиотека Tkinter.
2. Скачайте проект.
3. Запустите файл `weather_diary.py` командой: `python weather_diary.py`.

Примеры использования:
- Введите дату, температуру, описание и отметьте, есть ли осадки.
- Нажмите "Добавить запись" для сохранения.
- Используйте фильтры для поиска по дате или температуре.
- Нажмите "Сохранить в файл", чтобы сохранить все записи.
- Загрузите данные из файла через "Загрузить из файла"# sosal
